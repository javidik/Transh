# Профессиональный калькулятор траншевой ипотеки

Этот проект представляет собой профессиональный дашборд для расчета траншевой ипотеки с визуальными диаграммами, таблицами и графиками ежемесячных оплат.

## Функциональность

- Расчет ежемесячных платежей по аннуитетной формуле
- Интерактивный дашборд с визуализациями
- График ежемесячных платежей с обозначением интервалов траншей
- Диаграмма структуры платежа по траншам (круговая диаграмма)
- Диаграмма годовых выплат (столбчатая диаграмма)
- Диаграмма структуры общей суммы выплат (основной долг vs переплата)
- Таблицы с детализацией по траншам и ежемесячным платежам
- Сводные показатели (переплата, общая сумма выплат и т.д.)
- Интерактивные элементы с мгновенным обновлением результатов

## Дашборд включает:

- **Секция ввода параметров**: для настройки условий ипотеки
- **Сводные показатели**: основные метрики ипотеки
- **Графики и диаграммы**:
  - График ежемесячных платежей
  - Круговая диаграмма структуры платежа по траншам
  - Столбчатая диаграмма годовых выплат
  - Круговая диаграмма структуры общей суммы выплат
- **Таблицы**:
  - Детализация по траншам
  - Расчет ежемесячных платежей (первые 24 месяца)

## Использование

Для запуска калькулятора просто откройте файл `index.html` в любом современном браузере.

## Технологии

- HTML5
- CSS3
- JavaScript (без внешних библиотек)
- Canvas API для визуализации графиков
- Responsive дизайн для адаптации под разные устройства
