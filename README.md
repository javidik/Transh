# Калькулятор траншевой ипотеки

Современный калькулятор для расчета ипотеки с неограниченным количеством траншей. Приложение позволяет пользователям вводить параметры ипотеки, включая несколько траншей с разными суммами, ставками и датами выдачи, а затем визуализирует график выплат, структуру платежей и другую важную информацию.

## Особенности

- **Расчет траншевой ипотеки**: Поддержка неограниченного количества траншей с разными параметрами
- **Гибкие параметры**: Возможность выбора аннуитетных или дифференцированных платежей
- **Интерактивные диаграммы**: Визуализация графика выплат, структуры платежей и остатка долга
- **Детализация платежей**: Таблица с пошаговой детализацией всех платежей
- **Современный дизайн**: Минималистичный, футуристичный интерфейс с плавными анимациями
- **Адаптивность**: Корректное отображение на различных устройствах

## Технологии

- HTML5
- CSS3 (с использованием современных возможностей)
- JavaScript (ES6+)
- Chart.js для визуализации данных
- Responsive Design

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/mortgage-tranche-calculator.git
```

2. Перейдите в директорию проекта:
```bash
cd mortgage-tranche-calculator
```

3. Откройте `index.html` в вашем веб-браузере

## Использование

1. Введите общие параметры ипотеки:
   - Базовая процентная ставка
   - Общая сумма кредита
   - Тип платежа (аннуитетный/дифференцированный)
   - Срок кредита

2. Добавьте транши с индивидуальными параметрами:
   - Сумма транша
   - Процентная ставка
   - Дата выдачи

3. Нажмите "Рассчитать ипотеку" для получения результатов

4. Просмотрите дашборд с:
   - Общей информацией по кредиту
   - Графиками выплат
   - Структурой платежей
   - Таблицей детализации
